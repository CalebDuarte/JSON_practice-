1. What is JSON?

JSON stands for JavaScript Object Notation. It is a way to store and share data as text. It is easy for humans to read and easy for computers to process. Below is our car data as a JSON string — this is how a server would send it to us.

4. The Reviver Function

JSON does not have a Date data type. When we stringify a Date it becomes a plain string. When we parse it back it is still just a string — not a real Date. The reviver function is an optional second argument we pass to JSON.parse() to fix this — it lets us convert values as they are being parsed.
